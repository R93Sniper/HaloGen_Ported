# HaloGen_Ported
This repository contains the archived and organized source files for the cancelled Command & Conquer Generals: Zero Hour Total-Conversion Mod HaloGen alongside a port of those files for Halo Combat Evolved on The Master Chief Collection.


## Source Files
If you are simply looking for the original source files, check Source/Original and navigate to the file of choice. Files are included in their original archived versions (such as .gmax and .w3d) as well as modern formats (such as .max and .blend).

## Ported Files
If you are looking for the files that have been adjusted for use with Halo Combat Evolved, check Source/Halo1 and navigate to the file of choice. Unless specified, the models will not be modified in this state and mostly have additions (ie skeletons and collision meshes). These will almost exclusively be in .max format

## Modding Kit Files
If you are looking for the data/tags files, check Modding/data and Modding/tags for the relevant files.
